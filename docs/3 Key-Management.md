# Week 3 – Key Management

## Objective
Understand secure key generation, storage, and handling practices.

## Key Generation Methods
- **PRNG** – Software‑based randomness.
- **TRNG** – Hardware‑based randomness.
- **RSA Key Generation** – Large prime factorization.
- **ECC Key Generation** – Elliptic curve math.
- **KDFs** – Derive keys from passwords.

## Key Storage Methods
- Key Management Systems (KMS)
- Hardware Security Modules (HSM)
- Encrypted key files
- Password‑protected key files

## Best Practices
- Use strong randomness sources.
- Rotate keys regularly.
- Enforce least‑privilege access.
- Secure backups.
- Audit key access.

## Summary
Week 3 focused on ensuring that keys are generated securely and stored safely, as encryption is only as strong as its key management.
