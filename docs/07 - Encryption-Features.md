# Week 7 – Encryption Features

## Objective
Implement AES‑256‑CBC encryption and decryption using OpenSSL.

## Features Added
- `-e` (encrypt) and `-d` (decrypt) modes
- Key generation using `openssl rand`
- User‑provided key support
- Error handling for:
  - Incorrect keys
  - Corrupted files
  - Missing arguments

## Summary
Week 7 introduced the core encryption functionality, enabling the tool to securely encrypt and decrypt files.
