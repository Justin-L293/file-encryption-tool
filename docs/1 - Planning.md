# Week 1 – Planning

## Objective
Define the project scope, goals, and overall structure for the File Encryption Tool project.

## Tasks Completed
- Researched encryption basics, including symmetric and asymmetric encryption.
- Identified project goals and expected deliverables.
- Outlined the core features of the encryption tool.
- Drafted a 12‑week project timeline with milestones.

## Summary
Week 1 established the foundation of the project by clarifying objectives and determining the scope of work. This included understanding what the tool should accomplish, how it should function, and what security considerations would guide development.
