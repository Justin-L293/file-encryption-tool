# Flowchart Outline

## Overview
This document contains both flowcharts created during the development of the File Encryption Tool:

1. **Week 5 – Encryption Process Flowchart Outline**  
   A high‑level conceptual diagram created during the design phase.

2. **Week 11 – Full Encryption/Decryption Workflow**  
   A detailed, finalized flowchart representing the complete logic of the tool.

These diagrams support the design documentation found in  
**UI/UX Design** and  **Technical Specifications**.
